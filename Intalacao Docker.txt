Baixe Docker Desktop https://www.docker.com/products/docker-desktop/

Clonando o repositorio do supabase com o GitDesktop instalado abra cmd digite git clone --depth 1 https://github.com/supabase/supabase

abra a pasta cd supabase/docker

copie o .env.example para .env precisa editar o arquivo .env e o supabase/docker/volumes/api/kong.yml 
no .env edite 
POSTGRES_PASSWORD= N√ÉO USE @ NA SENHA

JWT_SECRET= 
ANON_KEY= 
SERVICE_ROLE_KEY=

no kong.yml consumers:

username: anon keyauth_credentials:
key: COLOQUE AQUI A ANONKEY
username: service_role keyauth_credentials:
key: COLOQUE AQUI A SERVICEKEY
